# Wine & Spirits Supplier Analytics Dashboard

## Overview
This Power BI dashboard analyzes supplier performance using a publicly available Wine & Spirits dataset. It’s designed to showcase advanced BI capabilities including strategic segmentation, dynamic commentary, and visual storytelling — ideal for decision-makers and recruiters seeking data-driven insights.

## Key Features
- **Top Supplier Identification**: Highlights the leading supplier based on YoY growth and total sales.
- **Supplier Benchmarking**: Uses scatter charts and quadrant logic to segment suppliers by growth and share.
- **Channel Segmentation**: Breaks down performance across C-STORE, GROCERY, DRUG, MASS, and MILITARY channels.
- **Sales Consistency Analysis**: Line charts track supplier stability and volatility over time.
- **Dynamic Commentary**: DAX-driven narrative updates based on slicer selections to guide stakeholder interpretation.
- **KPI Cards & Conditional Formatting**: Emphasizes key metrics with visual cues for rapid decision-making.

##  Strategic Insights
- Identifies high-growth suppliers and underperformers for targeted action.
- Reveals channel-specific trends to inform promotional strategy.
- Supports GRC and audit-readiness through transparent metric logic.
